# 퍼포먼스 마케팅 대시보드 설계 스펙

## 목표
퍼마 팀이 매일 아침 30초 안에 전날 성과를 파악하고, 주간 리뷰 때 팀이 함께 볼 수 있는 Streamlit 대시보드.

## 사용자 & 사용 맥락
- **사용자**: 퍼포먼스 팀 전체
- **사용 패턴**: 매일 아침 개인 체크 + 주간 리뷰 미팅 공용 화면
- **데이터 소스**: `raw/channel/YYYY-MM-DD.csv` + `raw/appsflyer/YYYY-MM-DD.csv`

## 탭 구조

### Tab 1 — 홈 (매일 첫 화면)

**목적**: 이 탭만 열면 오늘 뭐 챙겨야 할지 바로 파악

**구성**:
1. **날짜 선택기** + 새로고침 버튼 (기본값: 어제)
2. **KPI 카드 5개** — 총 비용 / ROAS / 플러스가입 / CAC / CVR (전일 대비 delta 표시)
3. **이상 징후 알림 박스**
   - 🔴 ROAS < 2.0 캠페인 목록
   - 🟡 ROAS 2.0~4.0 캠페인 목록
   - 🟡 전일 대비 비용 ±50% 이상 변동 캠페인
   - 이상 없으면 "이상 없음 ✅" 표시
4. **최근 7일 트렌드 라인 차트** — 지표 선택(비용/ROAS/가입/CAC)

**비즈니스 룰**:
- 네이버 브랜드KW(`NVR_CMP_01_브랜드KW`)는 ROAS 집계에서 자동 제외 (CLAUDE.md §6-1)
- 네이버는 채널 비교에서 `[자체매체]` 레이블 별도 표기
- ROAS 임계값: 🟢 ≥4.0 / 🟡 2.0~4.0 / 🔴 <2.0 (CLAUDE.md §5-4)

---

### Tab 2 — 채널·캠페인

**목적**: 채널 간 효율 비교 + 캠페인 랭킹으로 예산 의사결정

**구성**:
1. **필터**: 기간 / 채널 / 캠페인 목적
2. **채널 비교 바 차트** (지표 선택: 비용/ROAS/CAC/CVR)
   - 채널 브랜드컬러 적용: 구글 #4285F4 / 메타 #1877F2 / 네이버 #03C75A
   - 네이버는 `[자체]` 레이블, 브랜드KW 제외 표기
3. **캠페인 랭킹 테이블**
   - 컬럼: 캠페인명 / 목적 / 비용 / ROAS / CAC / 상태(🟢🟡🔴)
   - 기본 정렬: ROAS 내림차순
   - 클릭 시 해당 캠페인 소재별 성과 인라인 펼침

---

### Tab 3 — 소재

**목적**: 소재 타입·카테고리·AB 기준 성과 분석 → 소재 교체 결정

**구성**:
1. **필터**: 기간 / 채널 / 소재타입(IMG/VID/CRS/TXT) / 카테고리 / AB 여부
2. **소재타입별 ROAS 바 차트** (IMG vs VID vs CRS vs TXT)
3. **소재 랭킹 테이블**
   - 소재명 자동 파싱 → 타입 / 카테고리 / 시즌 / AB / 버전 컬럼 분리
   - 컬럼: 소재명 / 타입 / 카테고리 / 시즌 / AB / 비용 / ROAS / CAC / 상태
4. **AB 비교 뷰**: A/B 소재 짝 자동 매칭 → 나란히 지표 비교
   - AB 판정 기준: 노출 10만 이상 + 운영 7일 이상 (CLAUDE.md §6-4)

**소재명 파싱 규칙** (CLAUDE.md §3-3):
```python
# 예: VID_플러스멤버십_겨울_A_v2
parts = creative_name.split('_')
소재타입  = parts[0]          # VID
카테고리  = parts[1]          # 플러스멤버십
시즌     = parts[2]          # 겨울
AB여부   = parts[3] if parts[3] in ['A','B'] else None
버전     = parts[-1]         # v2
```

---

### Tab 4 — 퍼널

**목적**: 노출→구매 단계별 전환율로 병목 구간 파악

**구성**:
1. **필터**: 기간 / 채널 / 캠페인 목적
2. **퍼널 시각화** (수평 스텝 바)
   - 노출 → 클릭(CTR) → 설치CVR → 가입CVR → 구매CVR
   - 각 단계: 절댓값 + 전환율 + 전주 대비 delta
3. **채널별 퍼널 비교 테이블**
   - 컬럼: 채널 / CTR / 설치CVR / 가입CVR / 구매CVR / 가장 큰 이탈 구간(자동 감지)

- **숫자 포맷**: 금액 `₩1.2억`/`₩3,450만`, ROAS `4.52`, 퍼센트 `12.5%` (CLAUDE.md §8)
- **색상**: 채널 브랜드컬러 우선, 상태는 🟢🟡🔴

- **앱 실행**: `실행.bat` 더블클릭 (port 8502)
